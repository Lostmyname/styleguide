Align

Alignment classes follow the following syntax:

class - [media-query]

The following helper classes are available:

.align-left - Float an element left
.align-right - Float an element right
.align-none - Float an element none
.align-center - Set left and right margin to auto
.inline-align-center Align inline child to center
.inline-align-left Align inline child to left
.inline-align-right Align inline child to right

Examples

.align-left - align an element left
.align-center-on-sm - align anelement center on mobile
.inline-align-center-on-xs - align the inline contents of this element to center on portrait mobile
