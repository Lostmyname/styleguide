Display

Display classes follow the following syntax:

class - [media-query]

The following helper classes are available:

.display-block - Set elements display type to block
.display-inline - Set elements display type to inline
.display-none - Set elements display type to none
.full-width - Set elements width to 100%
.wide - Set elements width to 75%
.max-width - Set elements max-width to 100%
.full-height - Set elements height to 100%
